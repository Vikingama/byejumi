组件与类型：

  • 元素与组件：
    • 元素：构建 React 应用的最小单位，是 JS 中的普通对象，JSX 中使用的和 HTML 标签同名的元素并不等同于原生的 HTML 标签，这只是 React 内部抽象出来的一种标签的写法...
    • 组件：在 UI 界面中可以被独立划分、可重复使用、独立的模块，构建在元素的基础之上，类似于 JS 中对 function 的定义...
    • 组件会接受一个 props 参数，返回相应的 React 元素，将返回结果交给 ReactDOM 渲染出来...
  • 函数定义与类定义
  • 展示与容器
  • 有状态与无状态
  • 受控与非受控 --- 专门适用于 React 当中的表单元素...
  • 组合与继承
